# Аудио файлы для расследований

Сюда нужно положить аудио файлы:

## Красный лес
- `red-forest-ambient.mp3` — фоновая атмосферная музыка (рекомендуется 10+ минут)

## Рекомендации по форматам:
- **MP3** — лучшая совместимость
- **OGG** — хорошее качество при меньшем размере (не работает в Safari)
- **WAV** — без сжатия, большой размер

## Рекомендации по качеству:
- Битрейт: 128-192 kbps для баланса качества и размера
- Частота: 44.1 kHz
- Каналы: стерео

## Важно:
- Музыка будет играть в цикле (loop)
- Громкость по умолчанию 30%
- Fade in/out при старте/паузе
