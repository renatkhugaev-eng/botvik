# Аудио файлы для расследований

## Красный лес

### Основной слой (100% громкости):
- `follows-dark-ambient-194926.mp3` — атмосферная horror музыка

### Ambient слой (50% громкости, параллельно):
- `obemnyy-shum-pasmurnoy-pogody.mp3` — шум пасмурной погоды

## Рекомендации по форматам:
- **MP3** — лучшая совместимость
- **OGG** — хорошее качество при меньшем размере (не работает в Safari)
- **WAV** — без сжатия, большой размер

## Рекомендации по качеству:
- Битрейт: 128-192 kbps для баланса качества и размера
- Частота: 44.1 kHz
- Каналы: стерео

## Важно:
- Оба слоя играют в цикле (loop) параллельно
- Громкость по умолчанию 30% (основной) / 15% (ambient)
- Fade in/out при старте/паузе
